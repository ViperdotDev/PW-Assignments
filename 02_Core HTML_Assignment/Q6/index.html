1. HTML

    Primary Use Cases Key Features/Benefits
    Building the structure of web pages
    • Simple and easy to learn
    • Compatible with all web browsers
    • Allows for semantic markup

  2. CSS

    Primary Use Cases Key Features/Benefits
    Styling and designing web pages.
    • Separates content from presentation.
    • Provides flexibility through classes and IDs for styling.
    • Allows for responsive design with media queries.

  3. Javascript

    Primary Use Cases Key Features/Benefits
    Adding interactivity and functionality to web applications.
    • Runs on the client-side, reducing server load.
    • Supports asynchronous operations with promises.
    • Accesses the Document Object Model (DOM) for web page manipulation.
