1. Heading tag: Heading tags are used to render the headings of the content. There are 6 heading tags in HTML ranging from h1 to h6.
  2. Paragraph tag: Paragraph tags are used to display paragraph content.
  3. Title tag: Sets the title of the webpage, which appears in the browser's title bar or tab.
  4. Link tag: Specifies external resources like stylesheets and is commonly used to link to CSS files.
  5. a tag: Creates hyperlinks to other web pages or resources.
  6. Img tag: Embeds images on the web page
  7. ul tag: Defines an unordered (bulleted) list.
  8. li tag: Represents a list item within ul or ol lists, enclosing content to be listed.
